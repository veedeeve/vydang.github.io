# Python Practice Repository

This repository contains a collection of small-scale Python exercises designed to strengthen core technical skills in programming, data processing, and problem-solving. 

---

## Purpose  
The goal of this folder is to document my ongoing practice in Python through mini-assignments that highlight foundational programming concepts, data analysis techniques, and real-world scripting ability.

---

### **Assignment 01 — Word Count Functions**  
Developed two Python functions that operate on text datasets:

1. **Word Count Function**  
   - Returns the total number of words in a text file or string  
   - Demonstrates tokenization, cleaning, and string manipulation  

2. **Most Frequent Word Function**  
   - Identifies the most common word in the dataset  
   - Returns both the word and its frequency count

### **Assignment 02 — Top 3 Frequent Words**  
Created a function that:
- Parses the dataset  
- Identifies the **top three most frequently used words**  
- Returns the words and their counts  
- Demonstrates use of dictionaries, sorting, and text processing
